# CSS Grid component for layouts.

## Grid

```tsx
<Grid
  tag={'section'}
  rows={[0.2, 3, 0.1]}
  columns={[0.5, 1.5]}
  border={{
    bottom: true,
    left: true,
  }}
>
    ...
</Grid>
```

### Rows

Set the number of rows via and array of numbers. Each number represents a fraction.

### Tag

Pass in an html tag, defaults to `<main></main>`.

### Columns

Set the number of columns via and array of numbers. Each number represents a fraction.

### Border

Available options.

- top
- right
- bottom
- left

## Item

```tsx
<GridItem
  tag={'div'}
  rows={[1]}
  columns={[1, 3]}
  border={{
    bottom: true,
    left: true,
  }}
>
    ...
</GridItem>
```

### Rows

Pass in the row position of the item via an array of numbers, i.e. Row 1 sits a the top of the grid.

### Tag

Pass in an html tag, defaults to `<section></section>`.

### Columns

Pass in the column position of the item via an array of numbers, i.e. Spans Columns 1 and 2 ending at Column 3.

### Border

Available options.

- top
- right
- bottom
- left

### Import

```tsx
import { Grid, GridItem } from '@layouts'
```

```tsx
<Grid rows={[0.2, 3, 0.1]} columns={[0.5, 1.5]}>
  <GridItem
    rows={[1]}
    columns={[1, 3]}
    border={{
      bottom: true,
      left: true,
    }}
  >
      Header
  </GridItem>
  <GridItem
    rows={[2]}
    columns={[1, 2]}
    border={{
      left: true,
      right: true,
    }}
  >
    Sidebar
  </GridItem>
  <GridItem rows={[2]} columns={[2, 3]}>
    Content
  </GridItem>
  <GridItem
    rows={[3]}
    columns={[1, 3]}
    border={{
      top: true,
      left: true,
    }}
  >
    Footer
  </GridItem>
</Grid>
```
