# Loading

The **Loading** component is a simple loading indicator that renders a native `<span />` element with customisable styling. It supports colour variants and an optional centre alignment to position the loading indicator in the middle of its container.

## Usage

```tsx
import { Loading } from '@atoms/Loading/Loading.component'
```

```tsx
  <Loading
    color={'dark'}
    center={true}
  />
```

### modifiers

- Type: `string[]`
- Default: `[]`
- Description: Additional CSS classes for custom styling, appended to the component's base classes.

### color

- Type: `string`
- Default: `'dark'`
- Description: Specifies the colour variant of the loading indicator. This prop is used to apply the corresponding colour styling.

### center

- Type: `boolean`
- Default: None
- Description: When set to true, centres the loading indicator using a dedicated CSS class.

### ...rest

- Type: `Additional native HTML attributes for the <span /> element.`
- Default: None
- Description: Other HTML attributes can be passed to customise the `<span />` element.
