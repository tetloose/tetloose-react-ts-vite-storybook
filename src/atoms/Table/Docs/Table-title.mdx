# Table

The **TableTitle** component is used to render a table title cell for mobile devices. It wraps its content in a specialised `<Ti>` component and utilises the `<Typography>` component to display text, ensuring consistent styling for table headers.

## Usage

```tsx
import { TableTitle } from '@atoms/Table/_Table-title.component'
```

```tsx
  <TableTitle
    size={'body-lrg'}
    fontWeight={'bold'}
    text={'Title 1'}
    padding={{ default: 5 }}
  />
```

### modifiers

- Type: `string[]`
- Default: `[]`
- Description: Additional CSS classes for custom styling, which are combined with the default table title class.

### ...rest

- Type: `All other properties defined in TypographyProps, excluding tag, display, and richText`
- Default: None
- Description: Additional properties are passed to the `<Typography>` component, allowing further customisation of typography styling for the table title.
