# Checkbox

The **Checkbox** component is a versatile form control that renders an HTML `<input type={'checkbox'} />` wrapped in a custom label and spacer. It supports customisable styling, color variants, spacing, and error states.

## Usage

```tsx
import { Checkbox } from '@atoms/Form/_Checkbox.component'
```

```tsx
  <Checkbox
    padding={{ default: 2 }}
    color={'dark'}
    htmlFor={'accept'}
    id={'accept'}
    value={'true'}
    label={{
      text: 'Accept',
      size: 'body-med'
    }}
    error={!!acceptError}
    {...register('accept', { required: true })}
  />
```

### modifiers

- Type: `string[]`
- Default: `[]`
- Description: Additional CSS classes for custom styling. These modifiers are appended to the component's base classes.

### color

- Type: `string`
- Default: `'dark'`
- Description: Defines the color variant for the Checkbox. This is used to apply the appropriate color styling to the label.

### label

- Type: `LabelProps`
- Default: None
- Description: An object containing properties for the label. It typically includes the text and any typography settings required for the label.

### padding

- Type: `Spacers`
- Default: None
- Description: Spacing around the Checkbox component, applied via the wrapping Spacer component.

### margin

- Type: `Spacers`
- Default: None
- Description: Margin around the Checkbox component, applied through the Spacer component.

### error

- Type: `boolean`
- Default: `false`
- Description: Indicates whether the Checkbox is in an error state. When set to true, error styling is applied.

### ...rest

- Type: `InputHTMLAttributes<HTMLInputElement> (excluding the type property)`
- Default: None
- Description: Additional native HTML attributes.
