## Css Modules

The project makes use of CSS modules. The decision was taken to move away from styled components due to performance issues and problems with FOUC (flash of un-styled content).

The projects modules where possible are written in BEM structure for familiarity.

Css modules allows styles to be included using dot notation within the components. However due to using BEM we lose the Element and Modifier identifiers using dot notation.

For ease of development and aligning the styles we use CSS modules this way:

```tsx
import cs from "classnames";
import styles from "./Component-name.module.css";

export const ComponentName = ({
  variant,
  modifiers,
  ...rest
}: ComponentNameProps) => {
  // functions

  return (
    <div
      className={cs(
        styles["component-block"],
        styles["component-block--modifier"]
      )}
    >
      <span className={styles["component-block__element"]}>/* content */</span>
    </div>
  );
};
```

It is important to note that we use the `classnames` package as a way of joining the classes together.
